# ProxyPooler

## 简介

ProxyPooler是一个正在努力完成的、距离目标遥遥无期的、随便写写的、或许会长期维护的、基于Python的**代理**全自动化采集、校验、融合工具。本工具目前可以使用的主要功能为从互联网采集代理，但是本工具目的是打造一个支持自动化采集、校验、融合工具。

提示：此项目为我随便写的练习项目，早期写的，有些陈年老屎山代码是非常正常的。

## 已实现功能

1. 获取寥寥无几的几个国内网站的代理；
2. 简单的模块化；
3. 木有。
